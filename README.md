wm-validate
===========

Plugin jQuery para validação de formulários

## Instalação

    <!--  jQuery 1.7+  -->
    <script src="assets/js/jquery-1.9.1.min.js"></script>
    <script src="wm-validate/jquery.wm-validate-1.0.min.js"></script>

## Chamando o Plugin

    <script type="text/javascript">
	    $(document).ready(function(){
	        $('form').wmValidate();
	    });
    </script>
